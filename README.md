# About
What is this? This is a printer designed as a briefcase where the walls and Y-axis unfold to reveal a bedslinger 3D printer. The design objectives I gave myself when designing this printer were the following: It had to be the biggest possible printer which could fit as a carry-on personal item which can fit under the seat of an airplane.  
  
Why is it called Bruce? This printer was designed for Hack Club's Infill YSWS, where I have a $350 budget to build a printer with a unique element, and get sent to Rocky Mountian Rep Rap Festival to showcase it. Several of the other participants gave their printers random human names, so we figured it would be funny to get all of the printers name tags with these names. Hence, I created an acronym for my printer, and BRUCE was created.
